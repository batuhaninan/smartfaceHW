### Prerequisites
<div id="prerequisites"></div>

* [expo-cli](https://docs.expo.dev/get-started/installation/)
* npm

### Installation
<div id="installation"></div>

1. Clone the repo
```bash
git clone git@github.com:batuhaninan/smartfaceHW.git
```

2. Move to directory
```bash
cd smartfaceHW
```

3. Install npm packages
```bash
npm install
```

4. Run
```bash
npm start
```
